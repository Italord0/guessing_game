# Guessing game

- Este é um projeto console em Rust para adivinhar um número entre 1 e 100
- Criado a partir da [documentação oficial](https://doc.rust-lang.org/book/ch02-00-guessing-game-tutorial.html#generating-a-secret-number)

# Build

- Utilize o cargo para buildar e rodar o projeto

`cargo build && cargo run`
